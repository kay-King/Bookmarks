书签管理器

分类

您可以添加、编辑、删除和拖动排序书签。使用“导出书签”保存数据到 bookmarks.json、txt 或 html 文件，html 文件可直接导入浏览器。文件将保存到您的下载目录，请手动移动到程序所在文件夹与他人分享。导入时将文件放入同一目录并点击“导入书签”加载。

书签详情

使用帮助

程序介绍： 这是一个离线书签管理器，可帮助您在本地管理书签和留言，无需联网即可使用，支持分类管理和搜索。

功能：

  • 添加书签：点击“添加书签”按钮，填写名称、链接、简介和分类。
  • 编辑书签：点击“编辑书签”按钮，选择卡片上的编辑选项进行修改。
  • 删除书签：点击“编辑书签”按钮，选择卡片上的删除选项或勾选后点击“批量删除”，可通过“撤销”按钮恢复最近删除。
  • 拖动排序：按住卡片拖动到目标位置调整顺序。
  • 留言板：点击“留言板”按钮，双击文本区域编辑留言，保存后可在下次打开时查看。
  • 导出书签：点击“导出书签”选择保存为 JSON、TXT 或 HTML 文件，HTML 文件可导入浏览器。
  • 导入书签：点击“导入书签”加载 JSON、TXT 或 HTML 文件，重复书签和留言将被跳过。
  • 搜索书签：在搜索框输入名称进行筛选（支持防抖）。

注意事项：

  • 书签和留言数据存储在本地，清空浏览器缓存可能丢失数据，建议定期导出备份。
  • 导出文件默认保存到下载目录，需手动移动到程序文件夹以便导入。
